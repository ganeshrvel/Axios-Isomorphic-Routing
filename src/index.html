Axios Isomorphic Routing Controller for Node.js

For Server side Axios routing demo: /demo/server

For Client side Axios routing demo: fire up the browser console.

 

For indepth explanation go through README.md

src/examples/server.js for server side API calls

src/examples/client.js for client side API calls

 

src/api/consts.js for changing modifying constant values

src/api/end-points.js for modifying or adding API end points

 

express/routes.js for adding new server routes

 

Available API end points:

http://localhost:3001/cars?start=2005&end=2010

Method type: GET

 

http://localhost:3001/cities

Method type: POST

POST data: {"countries": "China", "limit": 2}

 

http://localhost:3001/try_put

Method type: PUT

POST data: {"param1": "12345", "param2": "abcdefg"}

 

http://localhost:3001/try_delete

Method type: DELETE

POST data: {"param1": "12345", "param2": "abcdefg"}

 

http://localhost:3001/test_cache1?param1=animals&param2=plants

Method type: GET

 

http://localhost:3001/test_cache2?param1=animals&param2=plants

Method type: POST

POST data: {"planet1": "mars", "planet2": "jupiter"}
